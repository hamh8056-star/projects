# Images AquaFish

Placez vos images d'aquaculture ici pour le slider.

## Format recommandé
- Format : JPG, PNG ou WebP
- Résolution : 1920x1080px (ou ratio 16:9)
- Taille : < 2MB par image pour de meilleures performances

## Utilisation
Pour utiliser vos images locales, modifiez le fichier `src/components/AquacultureSlider.tsx` :

```typescript
const aquacultureImages = [
  {
    id: 1,
    url: "/images/aquaculture/votre-image-1.jpg",
    alt: "Description de l'image",
    title: "Titre AquaFish"
  },
  // ...
];
```

## Exemples de noms de fichiers
- aquafish-bassin-1.jpg
- aquafish-monitoring.jpg
- aquafish-production.jpg
- aquafish-ferme.jpg
- aquafish-tracabilite.jpg
